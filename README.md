# Vintage National Semiconductor INS8060 (SC/MP) CPU Tube Labels

This repository documents vintage packaging and labeling of National Semiconductor INS8060 (SC/MP) microprocessors.  
Photos include original chip tubes, label variations, lot/date codes, and other historical packaging details.

## Purpose

To preserve and reference historical ephemera related to the SC/MP processor line, with a focus on:

- Label design and layout evolution
- Packaging material differences
- Date codes and part markings
- Distributor-specific labeling
